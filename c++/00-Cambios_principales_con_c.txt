POO

Clases - Describen una estructura
objetos / instancias - Un ejemplo en especifico de una clase 
instanciar - crear un objeto a partir de una clase
data members - variables
metodos - funciones
Constructores - Metodo especial dentro de una clase que inicializa los atributos de un objeto pudiendo (o no) asignarle valores iniciales y preparar el objeto para su uso

Abstracción - Concepto en el que reduces la complejidad de algo ocultando lo complicado (concepto caja negra)
encapsulacion - Otorgar acceso a datos privados desde interfaces publicas
herencia - se pueden crear clases hijas que heredarán propiedades de sus clases padre
polimorfismo - Se pueden tratar varios objetos de forma diferente a partir de su objeto base
