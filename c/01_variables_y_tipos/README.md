# Variables y Tipos de Datos en C

En C, es obligatorio declarar el tipo de dato antes de usar la variable.

## Lo que he aprendido:
- `int`: Para números enteros. Usa `%d` en printf.
- `float`: Para decimales. Usa `%f` en printf.
- `long`: Para enteros de 8 bytes
- `bool`: Para indicar true o false
- `char`: Para un solo carácter. Usa `%c`.
- `sizeof()`: Función útil para ver cuánta memoria ocupa una variable.
