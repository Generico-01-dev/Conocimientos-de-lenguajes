# Variables y Tipos de Datos en Java

En Java, es obligatorio declarar el tipo de dato antes de usar la variable.

## Lo que he aprendido:
- `int`: Para números enteros.
- `float`: Para decimales hasta 7 dígitos.
- `double`: Para decimales hasta 115 dígitos.
- `long`: Para enteros de 8 bytes.
- `boolean`: Para indicar true o false.
- `char`: Para un solo caracter.
- `String`: Para cadenas de caracteres. //No es un dato primitivo, sino un objeto
